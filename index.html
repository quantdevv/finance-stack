The protocol

Sequential by design

Each module assumes the ones below it. Time value of money shows up in every valuation; accounting underpins every analysis. We build the stack bottom-up so nothing is hand-waved.

You call the next module

This doc is the index. When you're ready, just say "give me M03" (or any module) and I'll generate its full standalone .html lesson. No rush, no autopilot.

Translated for an engineer

Where it helps, I'll lean on what you already know — systems, dependencies, data, state. Module 18 in particular is finance through a coder's lens: simulation, models, Python.

Understanding over coverage

The goal isn't to pass an exam — it's fluency. We go deep enough that financial news, balance sheets, and market talk stop being a foreign language.
