# Finance Stack

A 24-module, self-contained finance curriculum — from how money works to corporate finance, markets, valuation, derivatives, risk, quant finance, the CA/CFA paths, and personal finance. Every lesson is a single standalone HTML page (no build step, no dependencies).

**Start here → [`index.html`](index.html)** — the full curriculum map with links into every module.

## Structure

| Part | Modules |
|---|---|
| I — Foundations | M01–M04 |
| II — Commerce & Corporate Core | M05–M06 |
| III — Markets & Instruments | M07–M11 |
| IV — Analysis & Management | M12–M16 |
| V — Institutions & Systems | M17–M19 |
| VI — Credentials & Career | M20–M22 |
| VII — Capstone & Personal Finance | M23–M24 |

Each module page has Home / Prev / Next navigation, so you can read straight through or jump around from the index.

## Viewing it

- **Locally**: open `index.html` in any browser.
- **On the web / phone**: hosted via GitHub Pages — open the repo's Pages URL (Settings → Pages on GitHub, or check the repo "About" section for the link).
